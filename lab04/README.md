### Базовый датчик случайных чисел

**Задание:**
Реализовать базовый датчик случайных чисел.

### Решение:

В качестве основы генератора случайных чисел лежит *Мультипликативный конгруэнтный генератор*.

Алгоритм генерирует следующую последовательность чисел $X_{n+1}$ из предыдущего числа $X_n$ по следующей формуле:

$$X_{n+1} = (b \cdot X_n) \pmod m$$

Где:
*   $X_n$ — число, сгенерированное на предыдущем шаге ($n=0$).
*   $X_0$ — *Начальное значение* (seed). 
*   $m$ — *Модуль*. Определяет верхнюю границу диапазона генерируемых чисел.
*   $b$ — *Коэффициент умножения* (Multiplier). Должен быть целым числом.
*   $\pmod m$ — Операция взятия остатка от деления на модуль.

Параметры для генератора
*   $X_0$ = 1
*   $m$ = $2^{63}$
*   $b$ = $2^{32} + 3$

## 2. Метод середины квадрата 

Здесь \(k\) – количество цифр в начальном числе (чётное). Тогда:

$$
X_{n+1} = \left\lfloor \frac{ \left( X_n^2 \bmod 10^{2k} \right) }{ 10^{k} } \right\rfloor
$$

Алгоритмически:
1. Возвести \(X_n\) в квадрат: \(Y = X_n^2\).
2. Дополнить \(Y\) слева нулями до 8 цифр (если необходимо).
3. Удалить две крайние левые и две крайние правые цифры – оставить средние 4 цифры.
4. Полученное число — \(X_{n+1}\).
5. Для получения числа в интервале \([0, 1)\): \(\alpha_n = X_n / 10^{4}\).
