### Базовый датчик случайных чисел

**Задание:**
Реализовать базовый датчик случайных чисел.

### Решение:

В качестве основы генератора случайных чисел лежит *линейный конгруэнтный метод (ЛКМ)*.

Алгоритм генерирует следующую последовательность чисел $X_{n+1}$ из предыдущего числа $X_n$ по следующей формуле:

$$X_{n+1} = (a \cdot X_n) \pmod m$$

Где:
*   $X_n$ — число, сгенерированное на предыдущем шаге ($n=0$).
*   $X_0$ — *Начальное значение* (seed). 
*   $m$ — *Модуль*. Определяет верхнюю границу диапазона генерируемых чисел.
*   $a$ — *Коэффициент умножения* (Multiplier). Должен быть целым числом.
*   $\pmod m$ — Операция взятия остатка от деления на модуль.

Параметры для генератора
*   $X_0$ = 1
*   $m$ = $2^{63}$
*   $a$ = $2^{32} + 3$


Сравнение выборочного среднего:

* ЛКМ решение: 0.5013360442187095
* random.random: 0.5001442921162781
* Истинное значение : 0.5

Сравнение выборочной дисперсии:

* ЛКМ решение: 0.08330912175551902
* random.random: 0.08354241852749093
* Истинное значение: 0.08333333333333333(1/12)


### Выводы

**Достоинства:**

Экстремальная скорость: Требует всего несколько операций умножения, сложения и взятия остатка.

Простота реализации

**Недостатки:**

Предсказуемость: Зная несколько выходных чисел, легко вычислить параметры и предсказать все последующие.

Младшие биты: Младшие биты чисел, сгенерированных ЛКМ, имеют очень короткий период.
