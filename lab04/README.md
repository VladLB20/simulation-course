### Базовый датчик случайных чисел

**Задание:**
Реализовать базовый датчик случайных чисел.

### Решение:

В качестве основы генератора случайных чисел лежит *линейный конгруэнтный метод (ЛКМ)*.

Алгоритм генерирует следующую последовательность чисел $X_{n+1}$ из предыдущего числа $X_n$ по следующей формуле:

$$X_{n+1} = (a \cdot X_n) \pmod m$$

Где:
*   $X_n$ — число, сгенерированное на предыдущем шаге ($n=0$).
*   $X_0$ — *Начальное значение* (seed). Важно, так как от него зависит вся последовательность.
*   $m$ — *Модуль*. Определяет верхнюю границу диапазона генерируемых чисел.
*   $a$ — *Коэффициент умножения* (Multiplier). Должен быть целым числом.
*   $\pmod m$ — Операция взятия остатка от деления на модуль.

Полученное число $X_{n+1}$ является целым. Если нужны дроби (например, в диапазоне от 0 до 1), результат нормализуют:
$$result = X_n / m$$

Главная проблема ЛКМ - это его период. Рано или поздно последовательность начнет повторяться, так как количество возможных состояний ограничено значением. Максимально возможный период, при соблюдений условий **теоремы Халла-Добелле**, равен m.

В данной работе исопльзовались следующие значения параметров:
*   $X_0$ = 42
*   $m$ = $2^{63}$
*   $a$ = $2^{32} + 3$
